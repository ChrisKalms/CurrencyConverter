Currency Converter Help File

Getting Started

Connect to Data Source

Go to https://fixer.io/signup/free and fill in the prompts.
After getting your free API key, go to Settings -> API KEY, and add the key to the text field.

Getting Data

To load data from the api, open the menu and select the 'Update' button.
NOTE: The free fixer API only updates every hour, so to save you going over your monthly limit, the application will only allow you to update every 30 minutes. Trying to update more often will prompt a popup stating "Currencies are up to date".
After loading data, go to Settings -> Choose Conversion Time, and select the top result to bring the loaded data into the Application.
NOTE: The application saves the currencies each time you update. If for some reason you need to access a previous conversion time you may do so here and it will be loaded into the application for you.

Selecting Currencies

To select the relevant currencies to convert between, go to Settings -> Choose Currencies, and choose all of the currencies that you need to convert between.
NOTE: If there are no codes selected, the application will default to AUD and USD.

Converting

  1. Put value to convert into input field
  2. Select the code to convert from in first picker
  3. Select the code to convert to in second picker
  4. Select the Convert button
  5. Output field will be filled with the converted value

Troubleshooting

  • "No Data Available" : The database has not been filled, and so the application has no data to work with. You need to select 'Update' button in the menu.
  • "Cannot retrieve data" : The application is unable to get data. There are two possible causes for this issue.
    1. The API Key hasn't been added to the Settings or is wrong. You need to add or fix the API key so that it can connect to the data source.
    2. The application doesn't have access to the internet. Check your phones mobile data or wifi is connected.
